# CritterVitals

A calm, trustworthy mobile app for tracking the health of exotic pets — especially bearded dragons and rabbits.

Built with React Native + Expo following an "Editorial Mobile LIGHT" design system (warm earthy tones, generous spacing, premium but approachable feel).

## Features

- Daily logging (weight, symptoms, medications, husbandry)
- Medication reminders with smart scheduling
- Photo journal with side-by-side comparison
- Trend visualization and insights
- Beautiful on-device PDF vet reports with charts
- Clean, privacy-focused local storage

## Design

Follows the detailed Editorial Mobile LIGHT guidelines:
- Forest green primary brand (#2E5A3C)
- Warm terracotta accents
- Generous magazine-style spacing
- Playfair Display for headings, clean sans for data
- Large touch targets and calm interactions

## Getting Started

```bash
cd frontend
yarn install
yarn start
```

## Tech Stack

- Expo SDK 54
- TypeScript
- expo-sqlite (local database)
- expo-notifications
- expo-print + expo-sharing (PDF reports)
- react-native-gifted-charts

## Disclaimer

CritterVitals is a personal tracking and organization tool only. It is **not** a substitute for professional veterinary advice, diagnosis, or treatment. Always consult a qualified exotic animal veterinarian for any health concerns.

Built with care for proactive pet parents.
